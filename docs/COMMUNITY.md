# Community contributions

We welcome external contributors. A **community contribution** for program tracking is typically:

- A **merged pull request** from a GitHub user who is **not** listed as the primary maintainer, **or**
- A substantive issue/report that led to a merged fix (document the PR link).

## How to contribute

1. Read [`CONTRIBUTING.md`](../CONTRIBUTING.md).
2. Open an issue or small PR (docs, tests, UX, Soroban fixes).
3. Use the [pull request template](../.github/pull_request_template.md).

**Proof for auditors:** Link the merged PR from `main` in your submission pack; optional line in [`CHANGELOG.md`](../CHANGELOG.md) under **Community**.
